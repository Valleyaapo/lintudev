## 2025-05-24 - React to Vanilla JS Migration
**Learning:** For simple interactive components like contact forms in Astro, replacing React with vanilla JavaScript eliminates the entire React runtime from the client bundle, significantly improving performance without sacrificing functionality or maintainability.
**Action:** Identify other isolated interactive components (e.g., mobile menus, simple toggles) currently using React and refactor them to vanilla JS to further reduce bundle size.
